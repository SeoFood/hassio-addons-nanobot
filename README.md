# Nanobot Home Assistant Add-ons

Custom Home Assistant Add-on repository for Nanobot.

## Add repository to Home Assistant

Use this repository URL in Home Assistant Add-on Store:

```text
https://github.com/SeoFood/hassio-addons-nanobot
```

## Included Add-ons

- `nanobot`: Run the Nanobot AI agent as a Home Assistant add-on.
